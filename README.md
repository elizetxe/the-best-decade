# The Best Decade

A premium 60–90 second healthspan strategy game for [Regenerative Revival](https://regenerativerevival.com/).

**Ten years. Four dimensions. How well can you protect what matters?**

## Stack

- React 19 + TypeScript
- Vite 8
- Tailwind CSS 4
- No backend, no auth, no database
- `localStorage` for streaks, bests, badges, and preferences
- Seeded RNG for daily decades and friend challenges

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Engine tests

```bash
npx tsx src/lib/engine.test.ts
```
